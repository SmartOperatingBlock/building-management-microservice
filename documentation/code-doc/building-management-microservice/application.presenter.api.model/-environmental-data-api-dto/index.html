@Serializable
data class EnvironmentalDataApiDto(val temperature: ValueWithUnit<Double>? = null, val humidity: Double? = null, val luminosity: ValueWithUnit<Double>? = null, val presence: Boolean? = null)

Wraps all the environmental data associated to a Room. It corresponds to the model used with the API. So it describe:

  • the temperature inside the room

  • the humidity inside the room

  • the luminosity inside the room

  • the presence of someone in the room All the data may be not present.
